# Currículo - Yuri Teixeira

Este repositório contém o currículo profissional de Yuri Teixeira em formato LaTeX.

## Arquivo Principal
- `curriculo_yuri_teixeira.tex` - Arquivo LaTeX principal do currículo

## Como Compilar

### Requisitos
- Distribuição LaTeX (TeX Live, MiKTeX, etc.)
- Pacotes necessários:
  - inputenc
  - fontenc
  - babel
  - geometry
  - xcolor
  - titlesec
  - enumitem
  - fontawesome
  - hyperref
  - array

### Compilação
```bash
pdflatex curriculo_yuri_teixeira.tex
```

Ou se preferir usar XeLaTeX (recomendado para melhor suporte a fontes):
```bash
xelatex curriculo_yuri_teixeira.tex
```

### Online
Você também pode compilar online usando:
- [Overleaf](https://www.overleaf.com/)
- [ShareLaTeX](https://www.sharelatex.com/)
